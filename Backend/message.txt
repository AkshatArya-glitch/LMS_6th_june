DATABASE REQUIREMENTS:

Create or update the database tables needed for messaging.

Suggested table: messages

Fields:

* id INT AUTO_INCREMENT PRIMARY KEY
* sender_id INT NOT NULL
* sender_role ENUM('admin','user','trainer') NOT NULL
* receiver_id INT NOT NULL
* receiver_role ENUM('admin','user','trainer') NOT NULL
* subject VARCHAR(255)
* message TEXT NOT NULL
* conversation_id VARCHAR(100) NOT NULL
* is_read TINYINT(1) DEFAULT 0
* created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
* updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP

Suggested table: conversations

Fields:

* id INT AUTO_INCREMENT PRIMARY KEY
* conversation_id VARCHAR(100) UNIQUE NOT NULL
* participant_one_id INT NOT NULL
* participant_one_role ENUM('admin','user','trainer') NOT NULL
* participant_two_id INT NOT NULL
* participant_two_role ENUM('admin','user','trainer') NOT NULL
* subject VARCHAR(255)
* last_message TEXT
* last_message_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
* created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
